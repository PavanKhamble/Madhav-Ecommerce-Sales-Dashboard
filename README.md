# Madhav Ecommerce Sales Dashboard 🛒📊

## Overview

An interactive Power BI dashboard developed to analyze ecommerce sales performance, profitability, customer purchasing behavior, and payment trends. The dashboard provides valuable insights to support business decision-making.

## Key Metrics

* ₹438K Total Sales
* ₹37K Total Profit
* 6K Total Quantity Sold
* ₹121K Average Order Value (AOV)

## Features

* Sales Performance by State
* Monthly Profit Analysis
* Category & Sub-Category Performance
* Customer-wise Sales Analysis
* Payment Mode Distribution
* Interactive Filters for Quarter and State

## Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling
* Data Visualization

## Key Insights

* Maharashtra generated the highest sales.
* Clothing category accounted for the highest quantity sold.
* COD was the most preferred payment method.
* Monthly profit trends highlighted seasonal business performance.
